Image Sharing Platform (MERN Stack)

A full-stack Image Sharing Platform built using the MERN stack, allowing users to upload images with captions and view them in a dynamic real-time feed. The application focuses on efficient image handling, scalable backend architecture, and seamless frontend interaction.

🛠️ Tech Stack
Frontend: React.js (Vite)
Backend: Node.js, Express.js
Database: MongoDB
API Handling: Axios
Image Handling: Optimized storage & delivery

✨ Features
📸 Upload images with captions
📰 Real-time feed display
🔗 RESTful API integration
⚡ Fast and responsive UI using Vite
🗂️ Structured backend (MVC pattern)
🌐 Seamless frontend-backend communication



⚙️ Installation & Setup
🔧 Backend Setup
cd Backend
npm install
npm start

💻 Frontend Setup
cd frontend
npm install
npm run dev

🌍 Environment Variables

Create a .env file in the Backend folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection_string

🚀 Usage
Backend runs on: http://localhost:5000
Frontend runs on: http://localhost:5173
